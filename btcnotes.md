# Generating keys

- generate seed
  - generate random number
  - get hash from random
  - calculate seed words from (random+hash)
- seed to master
  - master private
  - master public
- derivation
  - `m/49'/0'/0'` (BIP49 P2WPKH-nested-in-P2SH / ‘3’ SegWit addresses)
  - `m/84'/0'/0'` (BIP84 P2WPKH / ‘bc1’ native SegWit addresses)
